PROYECTO GUAYAGAMER
Elaborado por: Santiago Ramos
Fecha: 02/03/2020
Proposito. Dado un archivo de texto con el detalle de las conferencias se se van a dictar, el programa calendarizará las conferencias en los salones necesarios.
Instrucciones: El siguiente es un programa que se puede ejecutar por consola y su salida se podra visualizar por texto. 
Para ejecutar el programa se debe usar el comando java -jar meetings.jar <ruta y nombre del archivo de texto que contiene las conferencias>
El archivo de entrada puede ser similar al archivo entradaConferencias.txt adjunto en este proyecto.
